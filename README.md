This is a learning repo for elixir graphl by impl a medium clone

Features

* Register and Sign In 
* JWT Token
* Forget Password
* Create a Post
* Add category to the post (many to many)
* Create comments
* Send email to the user on new comment
* Claps/upvote (Subscription)
* Deploy to Heroku
